# Unminimizer

Unminimizer is a macOS menu bar utility that allows you to unminimize windows using a keyboard shortcut.

## Installation

### Homebrew

```bash
brew install --cask unminimizer
```

### Direct Download

1. Download the latest release from [Releases](https://github.com/bjornorri/Unminimizer/releases)
2. Open the DMG file and drag Unminimizer to your Applications folder
